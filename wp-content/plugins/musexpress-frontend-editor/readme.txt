The first WordPress in-browser editor for Adobe Muse

== Description ==

Save time and make your life simpler with this powerful visual editor. Give your clients an easy to use tool to manage text, images and video themselves. Instantly!

= Edit, save and publish... in a few clicks! =
A CONTENT EDITOR ANYONE CAN USE
The fastest way to revise your website content. No need to load the admin backend. Make changes directly on your page and see how everything is updated in real time right before your eyes.

= The perfect Tool for your clients =
Let your clients make live changes themselves without ever needing to open up a setting panel.
 
= Text, Image and Video Editor =
Adjust the style of your site text with this visual tool. You can also add different tags (H1,H2 etc.). Change the site images is simpler than ever. Just upload a new one, select and publish in a few clicks! Choose the best video from your media gallery or add a new one and have it directly on your page in one click.

= Embed a Youtube Video =
With the Frontend Editor you can easily embed a Youtube video and resize it as you want in the page.

= Download MusexPress for free to have your MusexPress Frontend Editor =
You need to install the MusexPress plugin to use the MusexPress Frontend Editor. You can download it for free on [www.musegain.com/musexpress](https://www.musegain.com/musexpress).

== Installation ==

Follow these steps in order to make the plugin work:

1. Make sure that you have installed the MusexPress plugin. You can download it for free on [www.musegain.com/musexpress](https://www.musegain.com/musexpress)
2. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin directly the WordPress plugins screen.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the MusexPress->Settings screen to configure the plugin


== Frequently Asked Questions ==

= Minimum Requirements =
* PHP version 5.6.0 or greater (PHP 7.1 or greater is recommended)
* MySQL version 5.6 or greater
* MusexPress 3.1.2 or greater

= What is MusexPress ? =
The first and mighty solution to match Adobe Muse with WordPress.
MusexPress is a free WordPress plugin that allows you to design websites in [Adobe Muse](http://muse.adobe.com/) and manage any contents with WordPress.

= Where can I get support or talk to other users? =
If you get stuck, you can ask for help on the [MusexPress Plugin Forum](https://www.musegain.com/forums) or on the [Official MusexPress Facebook Group](https://www.facebook.com/groups/musexpress.cms/).
For any help with MusexPress premium plugins (payment products) on MuseGain, you can use our [helpdesk](https://www.musegain.com/help/).

= Where I can find a complete documentation and video tutorials ? =
You can find everything you need to start with MusexPress on our [documentation section](https://www.musegain.com/documentation) or you can learn how MusexPress works with our [video tutorials](https://www.youtube.com/channel/UCozlJtbWQao1N_ViGp7BmuA).

= How can I design my website in Adobe Muse and then use it with WordPress ? =
The only things that you need to do when your Adobe Muse website is ready to go online is to install the MusexPress plugin in WordPress, then upload the Muse website on your server and finally convert it with MusexPress. 
Thats' all!
